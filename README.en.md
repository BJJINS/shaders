# GLSL Shader Demos

This repository contains a collection of GLSL shader demos showcasing various graphics rendering techniques.

## Demo List

- `cloudy day.glsl` - Simulates a cloudy weather effect
- `rasterizationRenderer.glsl` - Rasterization rendering demo
- `raymarching.glsl` - Ray marching technique demo
- `raytracing.glsl` - Ray tracing technique demo
- `sdf.glsl` - SDF (Signed Distance Function) based rendering demo
- `toon shading.glsl` - Toon shading effect demo

## Usage

To run these shaders, you need an environment that supports GLSL, such as WebGL or an OpenGL context. You can use frameworks like Three.js, GLFW, or other GLSL-compatible libraries to load and execute the shader code.
