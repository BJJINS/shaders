# GLSL 着色器演示

这是一个包含多种GLSL着色器演示的仓库，展示了不同的图形渲染技术。

## 演示列表

- `cloudy day.glsl` - 模拟多云天气效果
- `rasterizationRenderer.glsl` - 光栅化渲染演示
- `raymarching.glsl` - 光线行进技术演示
- `raytracing.glsl` - 光线追踪技术演示
- `sdf.glsl` - 基于SDF（有符号距离函数）的渲染演示
- `toon shading.glsl` - 卡通着色效果演示

## 使用方法

要运行这些着色器，您需要一个支持GLSL的环境，如WebGL或OpenGL上下文。可以使用Three.js、GLFW或其他支持GLSL的框架来加载和运行这些着色器代码。
